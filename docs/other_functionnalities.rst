Other Functionnalities
======================

Aikit offers other functionnalities that might be usefull for the DataScientist job.

.. rubric:: Model Json

A function to back and forth between a json-like object and a sklearn-like model that can be fitted.
This can be used to store the definition of a complete model into a json file for your projects.
